3D Texture from Blender

  • Render a volumetric cloud in Blender
  • Take a screenshot of every Z-slice of the rendered cloud
  • Pack all 64 slices into a 64×64×64 3D texture sprite sheet
  • Import into Unity as a 3D texture — used as the density field for raymarching
Source: Want to get Blender clouds into #unity? — https://www.youtube.com/watch?v=IGx_I4nzfxQ
Rendering

Raymarching & Artifact Reduction

  • Raymarch against the 3D density texture generated from Blender
  • Cast a ray per pixel; accumulate density and light at fixed step intervals
  • Interleaved Gradient Noise (IGN) randomizes each step size, hiding banding artifacts from uniform stepping
Source: https://blog.demofox.org/2022/01/01/interleaved-gradient-noise-a-different-kind-of-low-discrepancy-sequence/
Rendering

Henyey-Greenstein Phase Function

  • Simulates directional light scattering through cloud water droplets
  • g > 0 → forward scattering (bright toward the sun)  |  g < 0 → back scattering  |  g = 0 → isotropic
  • Applied at every raymarch step to weight how much light reaches the camera
float henyey_greenstein(float cosTheta, float g) { float denom = 1.0 + g*g - 2.0*g*cosTheta; return (1.0 - g*g) / (4.0*PI * pow(denom, 1.5)); }
Source: https://omlc.org/classroom/ece532/class3/hg.html

Secondary Raymarch & Beer-Lambert

  • Secondary raymarch from each sample point toward the main light source
  • Currently supports 1 directional light source
  • Beer-Lambert Law calculates light falloff through the cloud volume
// Extinction along view ray: transmittance *= exp(-density * _StepSize); // Light shadow along light ray: lightAtten = exp(-shadowDensity * lightStep * _ShadowDensity);
Rendering

Final Color Accumulation

  • Final color = sum of all light reaching the camera not blocked by the cloud itself
  • Each step: inscattered light × phase function × shadow attenuation × transmittance
  • Alpha = 1 − transmittance — fully opaque cloud = zero transmittance
  • Result blended over scene colour via alpha compositing
float3 inscatter = lightColor * phase * density * shadowAtten; accum += inscatter * transmittance * (1-exp(-density*step)); transmittance *= exp(-density * step); float alpha = saturate(1.0 - transmittance);
